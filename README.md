# Aplicación Móvil – Expo React Native

Proyecto desarrollado con Expo, React Native, React Navigation y Firebase. Incluye navegación moderna, manejo de fechas, notificaciones y componentes visuales.

## Tecnologías
- Expo ~54
- React 19.1
- React Native 0.81
- Firebase 12
- React Navigation 7
- Date-fns
- ChartKit / React Native SVG

## Estructura

src/
  screens/
  components/
  navigation/
App.js

## Instalación
git clone https://github.com/tu-usuario/tu-proyecto.git
cd tu-proyecto
npm install
npm start

## Ejecutar
npm run android
npm run ios
npm run web
